## 🚀 快速启动

### 1. 准备工作
确保你安装了 Node.js (v18+) 和 pnpm。

### 2. 填填配置
确保 `.env` 文件就位，填入你的 **数据库连接** 和 **企业微信 Webhook**。

```bash
DATABASE_URL="mysql://..."
WEBHOOKS_URL="https://qyapi.weixin.qq.com/..."
```

### 3. 一键起飞
```bash
# 安装依赖
pnpm install

# 同步数据库结构
npx prisma db push

# 启动印钞机 🖨️
npm run start:dev
```
